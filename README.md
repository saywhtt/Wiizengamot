Android - application for viewing photos from [unsplash.com](https://unsplash.com)

#### *Using stack:* 

1. Android SDK
2. Kotlin
3. Retrofit2
4. JavaRx
5. Fragment Result API
6. WorkManager API & BroadcastReceivers (for notifications)
7. Picasso
8. Gson
9. Dagger2
10. Architecture MVP
